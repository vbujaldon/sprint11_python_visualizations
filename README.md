# Sprint 11 ‚Äî Python Visualizations & Power BI Integration

## Description

This repository contains the practical exercises developed for **Sprint 11 ‚Äì Data Visualization with Python**, focused on exploratory data analysis and visualization techniques using Python and their integration into **Power BI**.

The project covers the complete analytical workflow:

* Connection to a **MySQL database**
* Data extraction and transformation with **Pandas**
* Creation of statistical visualizations using:

  * Matplotlib
  * Seaborn
  * Plotly
* Interpretation of analytical results
* Integration of Python visualizations into **Power BI dashboards**

The dataset used corresponds to the transactional database created in **Sprint 4**.

---

## General Objectives

* Understand the main data visualization methodologies.
* Select appropriate visualization techniques depending on data type and analytical objective.
* Work with both **static** and **interactive** visualizations in Python.
* Integrate analytical workflows across **MySQL, Python and Power BI**.

* Create basic visualizations using Pandas.

  * Matplotlib

  * Plotly
* Analyse relationships between numerical and categorical variables.
* Implement visual analytics pipelines combining databases and BI tools.
* Transfer Python visualizations into Power BI using Python scripts.
